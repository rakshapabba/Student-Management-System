# 🎓 Student Management System

**Flask + SQLite Web Application**

A simple **Student Management System** built using **Python Flask, SQLite, HTML, and Bootstrap**.
This project allows administrators to **manage student records easily** through a web interface.

---

# 📌 Features

✅ Secure **Admin Login**
✅ **Dashboard with student statistics chart**
✅ **Add new students**
✅ **Edit / Update student details**
✅ **Delete student records**
✅ **Search students by name**
✅ **Responsive Bootstrap UI**
✅ **SQLite database (no installation required)**

---

# 🛠 Technologies Used

* **Python**
* **Flask**
* **SQLite**
* **HTML5**
* **Bootstrap**
* **Chart.js**

# ⚙️ Installation & Run

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/student-management-system.git
cd student-management-system
```

### 2️⃣ Install dependencies

```
pip install flask
```

### 3️⃣ Run the application

```
python app.py
```

### 4️⃣ Open in browser

```
http://127.0.0.1:5000
```
